# LSYN (Linear Syntax Node)

LSYN is a custom JSON-based file format that combines site data and rendering logic into one file.

> **Project Note:** I built this in like 10 minutes because I was bored. It really has no porous.

## Why use LSYN?
It's cool

## Deployment
This project is configured for [Vercel](https://vercel.com). The `vercel.json` file ensures that `index.lsyn` is served with the correct headers to be viewed as a website in any browser.
